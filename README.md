# Bank Management System in C

A simple yet powerful Bank Management System implemented in C language. This console-based application provides essential banking operations for account management.

## Features

- Create new bank accounts
- View account details
- Deposit money
- Withdraw money
- Check balance
- View all accounts
- Delete accounts
- Transaction history
- Secure account access

## Prerequisites

- GCC (GNU Compiler Collection) or any C compiler
- Windows/Linux/macOS operating system
- Basic understanding of C programming

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Daniish-Qureshi/Bank-Management-System-in-C-Language.git
   ```

2. Navigate to the project directory:
   ```bash
   cd "Bank Management System in C(OK)"
   ```

3. Compile the source code:
   ```bash
   gcc main.c -o bank_management_system
   ```

4. Run the program:
   - On Windows:
     ```bash
     .\bank_management_system
     ```
   - On Linux/macOS:
     ```bash
     ./bank_management_system
     ```

## Usage

1. Launch the program
2. Use the menu to navigate through different options
3. Follow the on-screen instructions for each operation

## File Structure

- `main.c` - Main source code file containing the program logic
- `README.md` - This file, containing project documentation
- `account.dat` - Data file where account information is stored (created automatically)
